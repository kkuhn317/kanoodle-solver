# kanoodle-solver

This website can solve multiple versions of the game Kanoodle. Or Lonpos, because they're both the same game. You can set restrictions like the pieces must or can't touch, and add blockers which act like filled spaces.

I was inspired by Danny Padilla Games on YouTube (https://www.youtube.com/@DannyPadillaGames), go watch his YouTube Shorts!

And I might as well mention I used Google Gemini to help with the coding.
